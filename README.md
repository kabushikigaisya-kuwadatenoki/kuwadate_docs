# ドキュメントの配置方法について

ドキュメントは基本的に.md方式で記述し、保管します。
リポジトリにAWSやNext.js、DjangoやDockerといったディレクトリを作成し、それぞれに関連するドキュメントをディレクトリ直下に追加していく方法を取ろうと考えています。
